## Static page builder
Easy static page builder with php

## Usage
```bash
php -S localhost:8000
```

## Shortcode
|Shortcode     | Usage |
|--------------|---------------|
| %WMI_STYLE% | Insert style
| %WMI_NAVBAR% | Insert navbar |
| %WMI_FOOTER% | Insert footer |
| %WMI_CONTACT% | Insert contact form |
